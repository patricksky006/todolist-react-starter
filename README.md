## 啟動專案 
打開終端機

將專案複製到本地:
```
$ git clone https://github.com/patricksky006/todolist-react-starter.git
```
進入專案資料夾:
```
$ cd todolist-react-starter
```
安裝npm:
```
$ npm install
```
啟動專案:
```
$ npm run start
```
成功時，終端機會自動打開網站。

欲結束使用:
```
ctrl + c
```
